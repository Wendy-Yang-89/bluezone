# 团结引擎适配OpenHarmony
1. 安装[团结引擎](https://unity.cn/tuanjie/releases)
2. 安装OpenHarmony Build Support
![Tuanjie Hub > Installs > Add modules](/imgs/2026-01-26/ggi8ZVM8SLLKcSb9.png)
![Select  OpenHarmony Build Support](/imgs/2026-01-26/FuxLZN9vMODKE92i.png)
3. 创建新工程，切换平台为OpenHarmony
![输入图片说明](/imgs/2026-01-26/ALYv1Di1tOM8AyVE.png)![](https://docs.unity.cn/cn/tuanjiemanual/uploads/Main/openharmony-switch-platfrom.png)
